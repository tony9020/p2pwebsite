## Launch your development entreprenuer career with Pet 2 Place UAE- 



### Getting started

1. Clone the project

```sh
$ https://github.com/Nababratadeb- /Pet 2 Place- .git
```

2. Install the dependencies:

```sh
$ yarn

# or

$ npm install
```

3. Run the project

```sh
$ npm run dev

# or to build

$ npm run build
```

## Deploy to Vercel

Use the [Vercel CLI](https://vercel.com/download)

```sh
vercel
